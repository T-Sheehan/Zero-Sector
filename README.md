# 🚀 Zero Sector

A fast-paced, roguelike-inspired top-down space shooter focused on skill-based combat, reactive dodging, and dynamic upgrade builds.

---

## 🎮 Overview

**Zero Sector** is a 2D top-down space shooter where players navigate procedurally generated combat sectors, survive escalating enemy threats, and evolve their build through upgrades and abilities.

Inspired by classic arcade shooters like *Bosconian* and modern roguelikes like *Hades*, the game blends fast, responsive gameplay with replayable progression systems.

---

## ⚡ Core Features

- 🕹️ **Precision Movement & Shooting**  
  Smooth 360° movement with responsive combat controls

- ⚡ **Perfect Dodge System**  
  Skill-based mechanic that rewards timing with powerful counterattacks

- 🔁 **Procedural Sector Generation**  
  Modular, replayable combat arenas with varied enemy compositions

- 📈 **Combo System**  
  Build momentum through consecutive hits to increase combat effectiveness

- 💣 **Secondary Weapons & Abilities**  
  High-impact tools for clearing enemies and controlling space

- 🛰️ **Decoy System**  
  Temporary AI allies that draw aggro and assist in combat

- 🧠 **Build Diversity**  
  Customize playstyle through upgrades, synergies, and abilities

- 💀 **Permadeath + Meta Progression**  
  Learn, adapt, and improve across multiple runs

---

## 🧠 Design Philosophy

Zero Sector is built around the idea that **player skill should matter just as much as progression**.

The game emphasizes:
- Reactive gameplay (perfect dodging, positioning)
- Risk vs reward systems (combo, aggressive play)
- Replayability through systems, not just content

---

## 🗺️ Gameplay Loop

1. Enter a procedurally generated sector  
2. Fight enemies and avoid incoming fire  
3. Build combo and earn currency  
4. Clear objectives  
5. Upgrade in safe zones  
6. Progress to harder sectors  
7. Repeat until death or boss milestones
